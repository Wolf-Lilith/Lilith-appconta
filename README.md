# Suíte de Gerenciamento Pessoal 📱

Uma solução centralizada e eficiente desenvolvida para Android, focada na otimização de rotinas diárias através de três pilares fundamentais de controle operacional e pessoal.

O projeto foi estruturado com foco em alta performance, interface limpa (focada em usabilidade) e modularidade de funções.

## 🛠️ Funcionalidades Principais

*   **📊 Gerenciador de Contas:** Controle financeiro integrado para monitoramento de despesas e receitas com organização visual clara.
*   **✅ Gerenciador de Tarefas:** Fluxo de trabalho objetivo (To-Do) para listagem, priorização e conclusão de atividades pendentes.
*   **⏰ Lembretes com Despertador:** Sistema de notificações temporizadas e alarmes para garantir a execução de compromissos críticos no prazo.

## 🧠 Arquitetura e Lógica Aplicada
O aplicativo foi desenvolvido utilizando boas práticas de desenvolvimento Android, garantindo que a interface do usuário (UI) seja leve e responsiva, separando as regras de negócio da camada de visualização.

*   **Linguagem:** Kotlin
*   **Ferramenta de Build:** Gradle (Kotlin DSL)
*   **Interface:** Material Design (Foco em legibilidade e contraste)
